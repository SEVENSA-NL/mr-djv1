# Z05–Z06 – NL-page skeletons & Z03 integratie

Deze kit levert:

- Z05: skeletons voor alle belangrijke NL-pagina's:
  - /nl
  - /nl/bruiloften
  - /nl/bedrijfsfeesten
  - /nl/feesten-overig
  - /nl/impressies
  - /nl/mister-dj
  - /nl/veelgestelde-vragen
  - /nl/contact

- Z06: integratiehooks voor Z03-formulieren (Availability, IntroCall, WeddingIntake,
  FairOffer, OpeningDanceMix) in deze pagina's.

De skeletons heten bewust `page.skeleton.tsx` zodat je agent veilig kan mergen of hernoemen.
